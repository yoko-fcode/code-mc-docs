# ターゲティングを実施する

![画像](/ja/images/targetings.PNG)

* ターゲティング条件には以下の設定項目があります。

## 設定項目

|条件 | 必須 | オプション | 設定項目|
|:------:|:------:|:------:|:------|
|ユーザー条件 | ○ |  | デバイス　/ 閲覧ページ数 / 訪問回数 |
|ページ条件|  |  | 対象ページ|
|スケジュール |  | |  配信期間 / 曜日・時間帯 |
|流入元 |  |  | 流入パラメータ / 参照元種別 / 流入元URL|
|表示制限 |  | |  一定期間の表示回数 / 閉じるボタンでクローズされた場合の表示停止期間|
|HTML |  |  | ページ内のHTML要素でターゲティング|
|コンバージョン |  | |  特定のコンバージョンをしたユーザーを対象/対象外にする|
|Cookie連携 |  |  | Cookieの内容を利用してターゲティング|
|GA連携 |  | ○ | Google Analytics(以下GA)のディメンション情報を利用してターゲティング|
|MA連携 |  | ○ | 貴社がご利用中のMarketing Automation（以下MA）ツールの情報を利用してターゲティング|

## ユーザー条件

![画像](/ja/images/targeting1.PNG)

### デバイス【必須】
対象とするデバイスを選択します。1キャンペーンにつき1種類のデバイスを選択できます。
* PCまたはSP・タブレットを指定

### 閲覧ページ数：最小・最大で指定
セッション内で、最小と最大で指定した範囲のページ数を閲覧したユーザーをターゲティングします。

**※値として0を指定した場合、該当項目は指定されていない扱いになります。**

### 訪問回数：最小・最大で指定
最小と最大で指定した範囲の訪問回数のユーザーをターゲティングします。訪問回数はすべての期間でカウントされます。

**※値として0を指定した場合、該当項目は指定されていない扱いになります。**


## ページ条件

![画像](/ja/images/targeting2.PNG)

クリエイティブの表示対象とするページを設定します。

### 対象ページ

* すべてのページ: このサイトで「CODE Marketing Cloud」を導入したすべてのページを対象とします。
* 一部のページ: 表示対象ページを指定します。

### 対象ページの指定

表示対象URLを指定するホワイトリスト型、表示対象外とするURLを指定するブラックリスト型の2つの指定方法があり、双方を併用できます。
左の入力欄にURLを入力し、右の条件欄で当てはまる条件を指定してください。

| 条件欄 | 一致条件説明 |
|:----:|:---- |
|と前方一致 | 前方一致するURLすべてが対象となります。|
|と後方一致 | 後方一致するURLすべてが対象となります。|
|とフレーズ一致 | 部分一致するURLすべてが対象となります。|
|と完全一致 | 完全一致するURLのみが対象となります。|
|と正規表現一致 | 正規表現一致するURLすべてが対象となります。|

## 表示制限
* 一定期間での表示回数や、閉じるボタンでクローズされた場合の表示停止期間を設定することで、キャンペーンの過剰な表示を制限できます。
* 複数指定した場合、OR条件となります。

![画像](/ja/images/targeting5.PNG)

### 表示上限
* ユーザー一人あたりのクリエイティブの上限表示回数を指定します。
    * セッション単位と、ユーザー単位の両方で設定できます。

### 表示回数
* ユーザー一人あたりの一定期間内の上限表示回数を指定します。

### 閉じるボタンをクリックした場合の表示停止期間
* ユーザーが閉じるボタンをクリックしてクリエイティブを閉じた場合に、キャンペーンを非表示にする期間を指定します。



## スケジュール
キャンペーンの配信スケジュールを設定します。
配信期間を指定した上で、「その中の特定の曜日だけ」「その中の特定の曜日かつ特定の時間帯だけ」といった設定も可能です。

![画像](/ja/images/targeting3.PNG)

### スケジュールの設定有無

* すべての期間 : 本番公開後に即時表示が開始され、手動で停止しない限り配信が継続されます。
* 配信スケジュールを設定する： 配信期間を限定します。

### スケジュールの設定

配信期間と曜日、時刻を指定します
* 配信期間を指定する： チェックを入れると開始・終了日時の入力フォームを表示します。
    * 配信期間は複数入力可能です。
* 曜日と時刻を指定する： チェックを入れると曜日と時刻の入力フォームを表示します。
    * 配信期間を複数設定した場合、ここで設定した曜日と時刻はすべての配信期間に適用されます。


## 流入元

流入元条件を設定できます。複数設定した場合、AND条件で扱われます。

![画像](/ja/images/targeting4.PNG)

### 流入パラメータ
* 流入元をパラメータで指定することができます。

左の入力欄にパラメータを入力し、右の条件欄で当てはまる条件を指定してください。

| 条件欄 | 一致条件説明 |
|:----:|:---- |
|と前方一致 | 前方一致するURLすべてが対象となります。|
|と後方一致 | 後方一致するURLすべてが対象となります。|
|とフレーズ一致 | 部分一致するURLすべてが対象となります。|
|と完全一致 | 完全一致するURLのみが対象となります。|
|と正規表現一致 | 正規表現一致するURLすべてが対象となります。|


### 参照元種別
* 参照元種別については以下が指定可能です。
* SSLページから非SSLページへの流入の際、検索キーワードが参照できない場合があります。

|参照元 | 例 |
|:----:|:----|
|直接流入 | リファラ無し。|   
|自然検索 | Google、Yahoo!、Bing、AOL、Ask.jp、Baidu、Yandexに対応。 |
|有料検索 | Google Analtyicsの判別ロジックに準拠しており、「utm_medium」の値に「cpc」「ppc」「cpa」「cpm」「cpv」「cpp」のいずれかが付与されている場合に判定。|
|ソーシャル流入 | Google Analyticsの区分に準拠（Facebook、Twitter、Youtube、Amebaなどからの流入を判別）。|
|その他サイト | 上記いずれでもない場合。|

### 流入元URL

* 流入元URLはドメイン部分だけでなく、URL全体で設定します。
* 流入元を指定する場合、左の入力欄にURLを入力し、右の条件欄で当てはまる条件を指定してください。

| 条件欄 | 一致条件説明 |
|:----:|:---- |
|と前方一致 | 前方一致するURLすべてが対象となります。|
|と後方一致 | 後方一致するURLすべてが対象となります。|
|とフレーズ一致 | 部分一致するURLすべてが対象となります。|
|と完全一致 | 完全一致するURLのみが対象となります。|
|と正規表現一致 | 正規表現一致するURLすべてが対象となります。|


## HTML
ページ内のHTML要素でターゲティングします。

![画像](/ja/images/targeting6.PNG)

### HTML要素によるターゲティング
* 利用しない：HTML要素によるターゲティングを行いません。
* 利用する：HTML要素によるターゲティングを行います。

### 条件設定
* 対象のHTML要素：jQueryのセレクタ形式で要素を指定。
* 検索文字列：HTML要素に含むテキストまたはvalue属性の値を指定。
* 判定方法欄では下記から当てはまる条件を指定してください。

| 条件欄 | 一致条件説明 |
|:----:|:---- |
|と前方一致 | 前方一致するURLすべてが対象となります。|
|と後方一致 | 後方一致するURLすべてが対象となります。|
|とフレーズ一致 | 部分一致するURLすべてが対象となります。|
|と完全一致 | 完全一致するURLのみが対象となります。|
|と正規表現一致 | 正規表現一致するURLすべてが対象となります。|


## コンバージョン
ユーザーの達成済みのコンバージョン種別やコンバージョン回数でターゲティングします。

![画像](/ja/images/targeting7.PNG)

### 対象ユーザー
* すべてのユーザー：コンバージョンによるターゲティングを行いません。
* 一部のユーザー：次に指定する条件指定によるターゲティングを行います。

### 条件設定
対象コンバージョンを指定するホワイトリスト型、対象外コンバージョンを指定するブラックリスト型の2つの指定方法があり、双方を併用できます。

![画像](/ja/images/targeting8.PNG)

左のプルダウンからコンバージョンを、右の入力欄にコンバージョン回数を指定してください。


## Cookie連携
サイトのCookie情報を利用してターゲティングします。
複数指定する場合、AND条件になります。

![画像](/ja/images/targeting9.PNG)

* Cookie名：参照するCookieの名称を指定。
* パターン：参照するCookieの値を指定。
* 一致形式欄では下記から当てはまる条件を指定してください。

| 条件欄 | 一致条件説明 |
|:----:|:---- |
|と前方一致 | 前方一致するURLすべてが対象となります。|
|と後方一致 | 後方一致するURLすべてが対象となります。|
|とフレーズ一致 | 部分一致するURLすべてが対象となります。|
|と完全一致 | 完全一致するURLのみが対象となります。|
|と正規表現一致 | 正規表現一致するURLすべてが対象となります。|

※一致形式は「フレーズ一致」または「正規表現一致」を推奨します。
